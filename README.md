## Requisitos 
- Docker

### Docker

Construir y ejecutar los contenedores:

```bash
docker-compose up --build
```

```bash
docker-compose down
```

## Variables de entorno

Para que el proyecto funcione correctamente, hagan un .env con estas cosas:

```
PORT=3000
NODE_ENV=development
```
